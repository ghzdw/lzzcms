field标签介绍
field格式1
格式: 
<zdw:field 属性名称="属性值"/>
可用属性 属性可用值
name* indexname(首页名字)、indextitle(首页标题)、indexkeys(首页关键字)、 indexdesc(首页描述)、powerby(版权)、record(备案号)
len 正整数，表示截取后长度
append 自定义字符串，表示最后附加的内容，当使用len发生截取行为后生效
sensitive 截取字符串的时候是否区分中英文，如果区分中文，则一个汉字视为2个长度，一个英文视为1个长度，比如"乐之者cms"，如果len="7", 则结果为"乐之者c";如果不区分，则中英文都视为1个长度。yes:区分 no:不分区，不设置该属性默认不区分中英文.
功能介绍: 获取网站首页的名字、首页标题、首页关键字、首页描述、版权、备案号
用法举例: 
比如：<zdw:field name="indexname"/>作用是取出后端配置的"主页名称"
field格式2
格式: 
<zdw:nav type="top">
   <zdw:field 属性名称="属性值"/>
</zdw:nav>
可用属性 属性可用值
name* url(栏目地址)、 personalstyle(栏目自定义样式)、 name(栏目名称)
len,append,sensitive 参见"field格式1"
功能介绍: 包含在nav标签中，获取栏目地址、栏目名称、栏目样式
用法举例: 
<zdw:nav type="top">
    <a href=" <zdw:field name='url'/> "><zdw:field name='personalstyle'/><zdw:field name="name"/></a>
</zdw:nav>
field格式3
格式:

len,append,sensitive 参见"field格式1"
功能介绍: 包含在doclist标签中，获取文档的标题,简短标题、点击量、发布日期等信息
用法举例: 
<zdw:doclist count="1" flag="h">
  <a href="<zdw:field name='docurl'/>"><zdw:field name="comm_title"/></a>
  <zdw:field name="comm_intro"/>
</zdw:doclist>

doclist标签介绍
doclist格式1
格式: 
<zdw:doclist 属性名="属性值">
   <zdw:field name="comm_title"/>
</zdw:doclist>
可用属性 属性可用值
count 表示要取出的内容的条数，正整数,默认10条
flag 内容的自定义标记,可用值为h(头条)、 c(推荐)、f(幻灯片)
sortby 以内容的什么属性排序:click(点击量)、 publishdate(发布日期)
sort 正序或者倒序,当sortby有值时，才起作用,默认asc:asc(从小到大)、 desc(从大到小)
columns 该属性只有doclist标签不被包含时才起作用。不指定该属性，默认获取的内容列表是当前正在生成的栏目;可指定栏目id表示从指定的栏目中获取内容列表。正整数(栏目id，可从后台栏目管理查看,多个使用"|"分割)、 all(不区分栏目，比如取点击量最高、头条、最新等时，如不需要区分栏目，则可如此指定)
功能介绍: 与field标签结合获取当前正在生成的栏目的文档列表
用法举例: 
<zdw:doclist count="1" flag="h">
 	<zdw:field name="comm_title"/>
	<zdw:field name="comm_intro"/>
</zdw:doclist>

choose标签介绍
choose格式
格式: 参照choose的用法举例
可用属性 属性可用值
无 无
功能介绍: choose、when、otherwise结合实现判断,choose为父标签，可以包含多个when， 但只能有一个otherwise,跟js里switch、case、default实现的功能是类似的。可以使用在relations、doclist、sonlist、clnlist 标签之中
用法举例: 
<zdw:relations>
    <zdw:choose>
        <zdw:when test="${index==2}">
            <span class="red"><zdw:relation name="comm_title" /></span>
        </zdw:when>							
    </zdw:choose>
</zdw:relations>

<zdw:doclist>
    <zdw:choose>
        <zdw:when test="${index==1}">
            <img src='<zdw:field name="comm_thumbpic" />'/>
            <span class="red"><zdw:field name="comm_title" /></span>
        </zdw:when>	
        <zdw:when test="${index==2}">
            <span class="green"><zdw:field name="comm_title" /></span>
        </zdw:when>	
        <zdw:otherwise>
            <span class="black"><zdw:field name="comm_title" /></span>
        </zdw:otherwise>							
    </zdw:choose>
</zdw:doclist>

功能介绍: choose、when、otherwise结合实现判断

relations标签介绍
relations格式
格式: 
<zdw:relations count="想要获取的相关文章的数量">
    <zdw:relation name="文章属性值"/>							
</zdw:relations>
可用属性 属性可用值
count 数字，表示想要获取的相关文章的数量
功能介绍: 只能在内容页使用，与relation合用获取与当前文章相关的文章
用法举例: 
<zdw:relations count="6">
	<zdw:relation name="index"/>							
        <a href='<zdw:relation name="docurl"/>'>
        	<zdw:relation name="comm_title"/>
        </a>
</zdw:relations>

relation格式
格式: 参考relations标签
可用属性 属性可用值
name index(相关文章的下标)、comm_title(标题)、docurl(访问路径)
功能介绍: 只能在内容页使用，与relations合用获取与当前文章相关的文章
用法举例: 参考relations

searchitem格式
格式: 
<zdw:results>

</zdw:results>

name* docurl(内容访问地址)、title(内容标题)、 intro(内容简介)
功能介绍: 结合results标签获取搜索结果
用法举例: 
<zdw:results>
   <a href='<zdw:searchitem name="docurl"/>'><zdw:searchitem name="title"/></a>
